### Possessive pronouns

| Pronoun | Possessive (m/f/n/pl) |
| ------------- | ------------- |
| я | мой / моя / моё / мои |
| ты | твой / твоя / твоё / твои|
| он | его |
| она | её |
| мы | наш / наша / наше / наши | 
| вы | ваш / ваша / ваше / ваши | 
| они | их |
