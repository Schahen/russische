### Possessive pronouns

| Pronoun | Possessive (m/f/n/pl) | Posession |
| ------------- | ------------- | ------------- |
| я | мой / моя / моё / мои | у меня есть |
| ты | твой / твоя / твоё / твои| у тебя есть |
| он | его | у него есть |
| она | её | у неё есть |
| мы | наш / наша / наше / наши | у нас есть |
| вы | ваш / ваша / ваше / ваши | у вас есть |
| они | их | у них есть |
